# How to Add Your Photo to the Portfolio

## Quick Steps:

### 1. Save Your Photo
- Take the photo you showed me (with the yellow neon background)
- Save it in this project folder (same location as index.html)
- Name it: **profile.jpg**

### 2. That's it!
- Refresh your browser
- Your photo will appear automatically in the About section

## Alternative Method:

If you want to use a different filename:

1. Save your photo with any name (e.g., `bhagya-photo.jpg`)
2. Open `index.html`
3. Find line 177: `<img src="profile.jpg"`
4. Change to: `<img src="bhagya-photo.jpg"`
5. Save and refresh
